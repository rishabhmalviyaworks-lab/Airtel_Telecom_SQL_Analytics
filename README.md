# Airtel_Telecom_SQL_Analytics
A SQL-based data analysis project on Airtel customer behavior, including payments, OTT subscriptions, recharge plans, call activity, and revenue insights. Includes optimized SQL queries and analytical results for real-world telecom use cases.

 **Overview**
This is a SQL-based data analysis project focused on Airtel customer behavior.
The project includes insights on customer payments, OTT subscriptions, recharge plan popularity, call activity, and revenue distribution.
All analysis is done using optimized SQL queries, making it a strong real-world telecom case study.

**Project Objectives**
1. Analyze customer payments and identify highest-paying users
2. Determine most popular recharge plans
3. Study OTT subscription behavior
4. Calculate revenue generated from different plans
5. Identify active customers based on call count
6. Find customers with longest average call duration
7. Detect customers without OTT subscription
